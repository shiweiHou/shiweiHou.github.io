leetcode70. Climbing Stars

3/Mar 2017

class Solution { public: int climbStairs(int n) { if (n <= 2) return n; // one 表示还有最后一步要走， two表示还有最后两步要走，我们从第3个台阶开始算，那么走到第三个台阶 // 如果是从第二个台阶也就是还有最后一步要走的时候，我们一共有两种办法：1 1 1 和 2 1 // 如果是从第一个台阶也就是还有最后二步要走的时候，我们一共只有一种办法: 1 2,所以初始化为 2 1 int one_last = 2; int two_last = 1; int all = 0; for (int i = 3; i <= n; ++i) { // 当我们走到第i个台阶的时候，从前面得到的结果，也就是第i-1 和 i-2 个台阶上走，一共有 one_last + two_last 种方法 // 当我们走过第i个台阶要准备走第i+1个台阶的时候，那么走到第i个台阶的方法就是前面得到all，这时候只需走一步就可以走到 // 第i+1个台阶，走到第i+1-2个台阶的方法，就是前面的one_last数值，所以要更新 two_last 和 one_last all = one_last + two_last; two_last = one_last; one_last = all; } return all; } };
