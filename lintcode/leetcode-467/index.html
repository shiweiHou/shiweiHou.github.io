leetcode 467.Unique Substrings in Wraparound String

25/Feb 2017

class Solution { public: int findSubstringInWraproundString(string p) { // 一开始想法是，构建一个哈希表，类型，发现不行，原因是string有可能很长而且每次都要进行查找操作，效率很低 // 其实最主要的是WA了，也没有找出原因 // 后来就想到，因为题目里要求每个字符串如果出现N次的话，只算1次，而且必须是挨着的。所以如果字符串 abcdefg 出现了，那么 // abcde 就没必要计算了，因为前一个字符串包含后一个。设一个大小为26位的数组，每个数组存的是以当前下标 + ‘a’ 的字符在p // 中出现的最大长度，最后将所有的加起来就行了，时间复杂度为O(N) vector letters(26, 0); int len = 0; for (int i = 0; i < p.size(); ++i) { int cur = p[i] - ‘a’; // 当前字符是哪一个 // 已当前字符为开始的最大顺序长度，没有就为1，因为本身也算1个 if (i > 0 && p[i - 1] != (cur + 25) % 26 + ‘a’) len = 1; else ++len; if (len > letters[cur]) letters[cur] = len; } int ret = 0; for (auto i : letters) ret += i; return ret; } };
