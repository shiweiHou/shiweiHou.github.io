LintCode 将表达式转换为逆波兰表达式

22/Dec 2016

问题描述： 给定一个表达式字符串数组，返回该表达式的逆波兰表达式。

问题分析：首先要搞明白什么是逆波兰表达式，wiki上面有很详细的介绍，题目意思就是将我们平常习惯的数学表达式表示成计算机所能理解的式子。因为人类可以很自然的理解例如1 + 2 * 3这样的式子，但计算机没有办法，计算机需要将其转换为123*+这样的形式才可以。

前面有一道题是将逆波兰表达式转换为中缀表达式，这道题刚好反过来。其实如果我们首先先自己在草稿纸上自己将中缀表达式转为逆波兰表达式的过程演算一遍，就很自然的明白转换规则：假设我们有一个vector ret用来专门存答案，有一个stack opera用来存操作符（+ - * / （） ），另字符串ch为表达式expression中的元素，规则如下：

  1. 假如ch为数字，那么很自然的，将其放入ret中
  2. 假如ch为操作符，那么：

  • 如果ch == "(",那么将ch放入到opera中；

  • 如果ch == ")",那么将opera中最顶层的（上面的所有操作符放入ret中

  • 如果ch 是四则运算符,那么根据此时opera中已有的字符的优先级，如果ch大于此时opera顶层操作符的优先级，那么ch放入到opera中，如果不是，那么将opera中的运算符弹出放入ret中，直到碰到第一个大于ch的优先级的运算符或者碰到第一个（或者走到头。

我们首先放进去一个字符@，表示优先级最低或者表示栈底 代码如下：

class Solution {
public:
    /**
     * @param expression: A string array
     * @return: The Reverse Polish notation of this expression
     */
    vector<string> convertToRPN(vector<string> &expression) {
        // write your code here
        vector<string> ret;
        stack<string> opera;
        int count = 0;
        int pos = 0;
        opera.push("@");
        while (pos < expression.size()) {
            string ch = expression[pos];
            if (ch != "+" && ch != "-" && ch != "*" && ch != "/" && ch != "(" && ch != ")") {
                ret.push_back(ch);
                ++pos;
                continue;
            } else {
                string cur_op = opera.top();
                if (cur_op == "@" || ch == "(") {
                    opera.push(ch);
                    ++pos;
                    continue;
                } else if (ch == ")") {
                    while (cur_op != "(") {
                        ret.push_back(cur_op);
                        opera.pop();
                        cur_op = opera.top();
                    }
                    opera.pop();
                    ++pos;
                    continue;
                } else {
                    if (cur_op == "(") {
                        opera.push(ch);
                        ++pos;
                        continue;
                    }
                    if (ch == "*" || ch == "/") {
                        while (cur_op == "*" || cur_op == "/") {
                            ret.push_back(cur_op);
                            opera.pop();
                            cur_op = opera.top();
                        }
                    } else while (cur_op != "@" && cur_op != "(") { // 加上cur_op != "("是为对 ( ( 1 + 2) )这种连续两个以上括号的情况处理
                        ret.push_back(cur_op);
                        opera.pop();
                        cur_op = opera.top();
                    }
                    opera.push(ch);
                    ++pos;
                    continue;
                } 
            }
        }
        string cur_op = opera.top();
        while (cur_op != "@") {
            ret.push_back(cur_op);
            opera.pop();
            cur_op = opera.top();
        }
        return ret;
    }
};
