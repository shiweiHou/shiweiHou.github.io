c++里一些类型之间的简单转换

30/May 2016

char -> string：

//stringstream 在头文件 sstream里面
char c;
string str;
stringstream stream;
stream << c;
str = stream.str();

char * -> string:

char *ch = "hello world";
string s(ch);

int <-> char:

int a;
char ac;
a = ac -'0';
ac = a + '0';

int -> string:

//第一种方法
int i;
string str;
stringstream stream;
stream << i;
str = stream.str();
//C++ 11推荐的方法
int i;
string str;
str = to_string(i);
//利用sprintf,buf先为一个字符数组，然后再直接转换为string
sprintf(buf, "%d", value);

string -> int:

string s = "345";
int i = stoi(s);

char * -> int

char *c = "hello";
int i = atoi(c);

大小写字符相互转换：

char c = 'A';
c = tolower(c);//c = 'a'
c = toupper(c);//c = 'A'

C++中string类型的大小写转换：

string s = "ABCD";
transform(s.begin(),s.end(),s.begin(),::tolower);// s = "abcd"
transform(s.begin(),s.end(),s.begin(),::toupper);// s = "ABCD"
